# React Expense App

```React tax calculator app.```

## Technologies
* HTML5
* CSS3 preprocessed with SASS 
* JavaScript
* ReactJS
* React Router
* Redux
* Redux thunk
* Firebase
* Jest for testing
* Enzyme for testing
* Webpack
* babel
* NodeJs
